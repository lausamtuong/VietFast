Simpler version of the 3 column layout that doesn't bother with the 100% height. This allows for the header and footer to have height:auto if required.

Any column can be the longest and the columns will equalise - what more could you ask for.

The left and right columns are coloured by the background of the container element. A different background colour can be applied to either column by using a repeating image as in my main example.

The left and right columns are a fixed width and the centre column is fluid.
